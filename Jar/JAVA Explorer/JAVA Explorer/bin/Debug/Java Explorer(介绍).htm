运行条件:

         系统中必须装有Microsoft.NET Framework 3.5 SP1或更高

(Vista中已安装Microsoft.NET Framework3.0但仍需更新，请到网上下载安装，如Microsoft或在百度中搜 Microsoft.NET Framework 3.5 SP1下载,后安装即可。)

 

功能:

     I.Java Explorer能在电脑中快速查看手机Java文件。

     II.Java Explorer能在电脑中运行手机Java文件。

        (备注:可更改运行程序)

     III.Java Explorer能在电脑中修改手机Java文件。

     IV.Java Explorer能在电脑中快速查看手机Java文件里内容。
