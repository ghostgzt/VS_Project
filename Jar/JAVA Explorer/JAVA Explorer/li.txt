    I.Java Explorer能在电脑中快速查看手机Java文件。
    II.Java Explorer能在电脑中运行手机Java文件。
    III.Java Explorer能在电脑中修改手机Java文件。
    IV.Java Explorer能在电脑中快速查看手机Java文件里内容。
